Valores que puede adoptar "type"

Etiqueta Valores ¿Para que sirve?
Type text Permite al usuario ingresar y editar texto
Type Email Se utliza para escribir un email
Type Submit Se utiliza para enviar un formulario al servidor para su procesamiento
Type Password Sirve para que las contraseñas no se vean
Type Reset Se puede resetear el formulario
Type Radio Sirve para poner diferentes obsiones
Type Checkbox Es igual al radio solo que cambia el diseño y selecciona varias obsiones
Type Button Es un boton
Type Color Nos permite seleccionar un color
Type Date Nos sive para seleccionar fechas
Type Datetime-local Selecciona la hora (aparte de fechas)
Type File Nos permite eleguir un fichero
Type Month Nos sive par selccionar un mes
Type Number Nos permite eleguir un numero
Type Range Nos permote seleccionar un rango ente 0 y 50
Type Search Es igual al de tipo texto solo que tiene un icono que permite elimar el texto
Type Tel Se utiliza para numeros de telefono
Type Time Se utiliza para poner horas
Type Url Se utiliza para digitar una url
Type Week Se utiliza para seleccionar una semana
